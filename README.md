## 🔧 Instalação

Clone o repositório:
```
git clone https://github.com/saraferreira10/pizza-palette-react-query
```

Acesse o diretório principal:
```
 cd pizza-palette-react-query
```

Faça a instalação:
```
npm install
```

Inicie o projeto:
```
npm run dev
```
